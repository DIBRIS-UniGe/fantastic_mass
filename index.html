Fantastic MASs and where to find them

What

This repository contains links to publicly available software developed inside the multiagent systems research area: frameworks for building MASs, add-ons and extensions of such frameworks, and implemented MASs.

Why

Via this “survey from the contributors” we aim at:

  • Collecting a large set of publicly available MASs to create a shared benchmark of MASs to be used by colleagues working in the (agent-oriented) software engineering, runtime verification, code analysis areas, etc.
  • Performing an analysis of the large set of MAS development and simulation frameworks along with their add ons, libraries, extensions, and making them also available in the public repository.
